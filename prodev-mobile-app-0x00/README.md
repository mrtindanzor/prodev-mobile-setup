# First Mobile App with React Native

## Scaffolding 

- Create an application using pnpx create-expo-app@latest .

- To start the app, run pnpm start

### Reset Project

Resetting the project helps remove all the bloatcode that come with an install to give you a clean working slate. Simply run pnpm reset-project. This move your already existing code to a subfolder with name of your project.
